How to Use

Search Mode

  1. Enter a target distance in meters (e.g., 150m)
  2. Adjust tolerance if needed (default 0.5m allows for slight variations)
  3. Click "Find Distances" to see all marking pairs that match
  4. Click any result row to highlight it on the track
  5. Use pagination to browse through results

Manual Selection Mode

  1. Click directly on any marking on the track to select it
  2. Click a second marking to see the running distance between them
  3. Click a third marking to replace the first selection
  4. Click a selected marking again to deselect it
  5. Use "Clear Selection" to start over

Example Use Cases

  • Training reps: Find where to start a 150m acceleration on the track
  • Workout planning: Locate 200m segments that don't start at the standard 200m line
  • Race preparation: Understand distances between relay zones and other markings
  • Coaching: Identify landmarks for specific training distances

Tips

  • Double-click the track to reset the view
  • Use mouse wheel or pinch to zoom in/out
  • Drag to pan around the track when zoomed in
  • Toggle water jump position for steeplechase markings
